# Summary

The purpose of this project is to build a customer-facing website for an esthetician.
The website is built using Vite, and served statically using GitHub Pages.
The website is written in React.js. It leverages Material UI (see https://mui.com/material-ui/).
The website relies on the static JSON data defined in the /content directory to dynamically render components with the JSON data

## Optimized Design

The website is designed and optimized for the following:
- keyword-optimized meta descriptions: SEO (using react-helmet-async)
- responsive experience: mobile is fully supported
- quick load times
- accessibility: the website uses colors and design styles accessible to all users
